Task 4 - Rock Paper Scissors

A simple command line Rock-Paper-Scissors game where the user plays against the computer.

Features:
--> User plays multiple rounds against the computer.
--> Tracks and displays scores.
--> Accept: 'rock', 'paper', 'scissors', or 'exit'.
--> Handles invalid input.

How to run:
1. Ensure Python 3 is installed.
2. Open terminal in the script folder.
3. Run the script.
